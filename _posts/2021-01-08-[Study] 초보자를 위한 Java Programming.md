# [Study] 초보자를 위한 Java Programming

# 11장 쓰레드

* 프로세스 :: CPU가 실행 중인 프로그램(명령어) : 구성요소 = 명령어+데이터
* 멀티 프로그래밍 :: 하나의 CPU가 여러 프로그램을 동시에 실행하는 것
* 멀티 프로세싱 :: 2개 이상의 CPU에 의해 여러 개의 프로그램이 동시에 실행되는 것
* 멀티 태스킹 :: 멀티 프로그래밍과 비슷한 개념으로, 여러 개의 task를 동시에 실행하는 것
