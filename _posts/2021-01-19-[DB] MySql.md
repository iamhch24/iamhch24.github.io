# DB

### 0 사용대상
* 디비는 연결과 관리만 하고
* 결국 사용자가 사용하는 것은 테이블임

### 1 권한
* Connect 권한 :: 접근 connect 권한, 세션권한
* Resource 권한 :: 리소스 생성, 수정, 삭제 할 수 있는 권한
* Admin 권한 :: 관리자 권한
